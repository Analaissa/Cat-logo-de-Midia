** como projeto funciona **
Aplicação desenvolvida para listar, visualizar e organizar filmes de forma simples e intuitiva. O sistema permite navegar por títulos, visualizar detalhes e gerenciar uma lista personalizada de filmes.

** Tecnologias Utilizadas **
Frontend: React.js
Estilização: CSS / Styled Components
Consumo de API: Axios
Backend (opcional): Node.js + Express
Banco de Dados (opcional): MongoDB / PostgreSQL
Gerenciamento de Pacotes: npm
